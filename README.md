# Путешествие по России (russian-travel)

### Обзор

Проект о путешествиях по России. Представляет из себя отзывчиво-адаптивный сайт выполнененный по БЭМ-методологии.

**Используемые технологии**

* отзывчиво-адаптивная верстка
* grid-layout
* flex-box
